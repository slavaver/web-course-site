**Цель:** подготовить окружение для работ, ознакомиться с основами фреймворка Vue в связке с TS на примере серии несложных задач.

## Подготовка

1. В документации Vue ознакомьтесь с разделом [Основы](https://ru.vuejs.org/guide/introduction.html). Предпочитаемый стиль API Composition.
1. Пройдите [официальный интерактивный учебник по Vue](https://ru.vuejs.org/tutorial/). Предпочитаемый стиль API Composition и SFC.
1. Для знакомства с Typescript рекомендуется пройти [курс TypeScript](https://code-basics.com/ru/languages/typescript) на code-basics.
1. О TS лучше читать первоисточники в виде [The TypeScript Handbook](https://www.typescriptlang.org/docs/handbook/intro.html). Если сложности с английским, то можно начать с [серии статей Изучение TypeScript](https://habr.com/ru/articles/663964/) на habr.
1. Проверьте наличие node.js, если его нет, то установите [Node.js](https://nodejs.org/en) четной версии. На главной странице есть варианты на выбор операционной системы и способа установки.
1. Проверьте установлена ли система контроля версий git и настроена ли она (имя, почта).

## Выполнение

1. Примите задание на [Github Classroom](https://classroom.github.com/a/dtdGBsWu) и клонируйте к себе репозиторий.
1. Перейдите в созданную при клонировании папку в терминале и установите все зависимости командой `npm install`. Если возникают ошибки, в первую очередь убедитесь, что в терминале вы находитесь в нужной папке.
1. Запустите проект командой `npm run dev`. И откройте страницу в браузере.
1. Изучите состав проекта и станицы в браузере. Потребуется кратко ознакомиться с [маршрутизатором vue](https://router.vuejs.org/guide/). Конфигурация лежит в папке router, в App.vue находятся 2 основных компонента `RouterLink` и `RouterView`, а сами отображаемые компоненты в папке views.
1. Для выполнения 1-й работы рекомендуется в Lab1View.vue импортировать SFC Vue c решениями задач из папки components/Lab1. Для задач у вас должны быть написаны функции по заданию и затем функция используется для реактивных вычислений. Для функций выполняющих условия задачи аннотируйте типы параметров и возвращаемых значений.
1. После решения всех задач в компонентах и подключении их к Lab1View.vue. Проверьте что все работает как ожидается в браузере.
1. Если все в порядке. То пробуйте собрать проект командой `npm run build`.
1. Если проект собирается то дополнительно проверьте код статическим анализатором командой `npm run lint`. Устраните ошибки, если они есть.
1. Отформатируйте код командой `npm run format`.
1. После этого сделайте коммит и отправьте его на ваш репозиторий в Classroom.

## Требования

- Все задачи имеют решение и требуемый в них вывод в интерфейсе через формы.
- Проект собирается, отформатирован и нет замечаний от ESLint и tsc, нет ошибок в консоли браузера.

## Задачи

### Задача Время прибытия

Дается два числа: час прибытия по расписанию и количество часов опоздания, рассчитайте новое значение прибытия.

Время прибытия по расписанию и новое указывается в 24-часовом формате. 24 час записывается как 0 (22 + 2 = 0)

### Задача Подпись

Дается строка с фамилией именем и отчеством, верните инициалы с фамилией.

### Задача Длина слова

Дается строка состоящая из слов и пробелов, верните длину последнего слова в строке.

### Задача Скобки

Дается строка содержащая только символы '(', ')', '{', '}', '[' и ']', определите, валидна ли она.

Входная строка валидна, если:

- Открытые скобки должны быть закрыты скобками того же типа.
- Открытые скобки должны закрываться в правильном порядке.
- Каждой закрывающейся скобке соответствует открытая скобка того же типа.

### Задача Тип треугольника

Дается 3 значения длин сторон. Вернуть тип треугольника: разносторонний, равнобедренный, равносторонний и невозможный, если нарушается неравенство треугольника.

### Задача Сложение векторов

Написать функцию, которая принимает два и более кортежей двумерных векторов и возвращает результирующий вектор, который является их суммой.

### Задача Расстояние между векторами

Дается два n-мерных вектора, рассчитать евклидово расстояние между ними.

### Задача Перемножение матриц произвольного размера

Дается две матрицы произвольного размера, проверьте возможность операции умножения и верните произведение этих матриц.

### Задача CIE XYZ в sRGB

Дается кортеж координат цвета в пространстве CIE XYZ в диапазоне [0, 1], используя формулу из [статьи sRGB](https://ru.wikipedia.org/wiki/SRGB) рассчитайте значения цвета в пространстве sRGB. Используйте функцию из прошлой задачи.

### Размеры по Золотому сечению

Используя литеральные типы и шаблонный литеральный строковый тип, создайте псевдоним типа, который позволяет указать размер css до 21 и только в em или rem. Например, 3rem, 21em. Напишите функцию, которая принимает число и опциональный параметр, а возвращает значение ближайшее к одному из чисел последовательности с em, если дан опциональный параметр то с rem.

### Задача Конвертация температуры

К функции из задачи прошлого семестра convertTemp добавьте аннотации типов параметров и возвращаемого значения. Вам пригодятся типы с литеральным значением, шаблонные литеральные строковые типы, перегрузка функций.

**Пример работы:**

```js
convertTemp(32, 'toC') -> '0 C'
convertTemp(10, 'toF') -> '50 F'
```
